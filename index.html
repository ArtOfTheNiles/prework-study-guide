HTML
  • The "head" element contains information about the webpage
  • The "body" element contains information shown to the viewer
  • The "ul" element is for Unordered Lists
  • The "li" element is for Line Items

CSS
  • CSS is used to define visual elements consistently across web pages
  • CSS properties can be overridden by elements later in the execution order
  • Hypothesis: properties should be ordered from broadest "*" to most specific ".card"
  • Variables can be defined in the :root{} structure and used multiple times

Git
  • All Git commands must start with "git"
  • "checkout" moves the working branch to the specified branch
  • "checkout -b" flag creates a new branch
  • "git add -A" adds all changes to the branch
  • "git commit -m "message" " commits the changes while -m adds the message
  • It is best practice to pull from main to ensure you're not working on outdated code

  • JavaScript and C# are remarkably similar
